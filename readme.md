# ✨ dressup

一个纯前端换装拼图工具。

## 🚀 在线使用
访问：[https://Seaout.github.io/dressup](https://Seaout.github.io/dressup)  
**注意**：需要自己准备素材 ZIP 包，或下载下方的示例包。

## 📦 素材 ZIP 格式

### 单人
角色名称.zip
├── 1头发/
│ ├── 1发型.png
│ └── 2发型.png
├── 2上装/
│ └── ...
└── (可选) preset.json


### 多人
多人.zip
├── 人物A/
│ ├── 头发/
│ ├── 上装/
│ └── preset.json
├── 人物B/
│ └── ...


详细帮助请点击页面左上角的 `?` 按钮查看。

## 📥 示例素材
下载 [示例](https://github.com/Seaout/dressup/releases) 体验。

## 🛠️ 本地运行
1. 克隆仓库
2. 用浏览器打开 `index.html` 即可。
